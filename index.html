INTRODUCTION OF WEBSITE

Implementation Of Gamification Elements

There are four main gamification mechanics which is Levels, Points, Badges and Leaderboard beside elements from dynamics and Aesthethics from MDA framework which is Unlockable, competition, profile icon, username, User interface and User experiences

"This gamification elements meant to bring more enjoyable feeling in using this Learning Management System (LMS)"

LEVELS

The starting point of any success story is knowing your current position in the business environment

POINTS

tracking the quizzes fulfilment, determinants for the leaderboard ranking and bench mark for unlocking the next level of the quiz.

PROGRESS BAR

User can track their course completion by percentage

BADGES

badges are given to user that score specific point for each level of quiz which are beginner, intermediate and expert.

LEADERBOARD

Display the ranking, username, latest points and badges

USERNAME

User can create username during sign up and edit the username in user profile

PROFILE ICON

User can change the profile icon in the user profile

UNLOCKABLE

Unlockable are implemented in the level, user need to complete the beginner level to unlock intermediate, and complete intermediate to unlock expert level and earn badges.

COMPETITION

Implement through competition in leaderboard to reach the highest ranking.

UI AND UX

The use of button, navigation bar, user-friendly forms, icons. dashboard, typography and color scheme for UI. The use of gamification and responsive design for the UX.
